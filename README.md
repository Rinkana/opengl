# opengl

OpenGL library that extends [lib_gl](https://github.com/Rinkana/lib_gl) to make it easier to use.

## Installation

Add this to your application's `shard.yml`:

```yaml
dependencies:
  opengl:
    github: Rinkana/opengl
```
